Local-only tool

PGP Decryptor

This page is intended to run locally and decrypt armored PGP messages in your browser using your private key and passphrase. It does not upload anything or store your private key.

How to use this safely

Keep this page local. Keep openpgp.min.js local. Do not host this publicly. Do not paste your private key into random websites.

If double-clicking this HTML does not work, run it via localhost instead.

Load your private key

Paste your armored private key, or upload the private key file you downloaded earlier.

Paste the armored encrypted message you want to decrypt.

Notes

This version is intentionally narrow:

- decrypt armored messages
- no signature verification
- no storage
- no network calls

That is the safer default.
